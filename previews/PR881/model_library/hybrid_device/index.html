HybridSystem

PowerSystems.HybridSystem — Type

Representation of a Hybrid System that collects renewable generation, load, thermal generation and storage.

Each generator is a data structure that is defined by the following components:

  • ThermalGen
  • Load
  • Storage
  • RenewableGen
